ETIQUETAS SEMANTICAS CSS

Las etiquetas semánticas tienen los siguientes posicionamientos en los cuadros. CSS nos ofrece, además, la posibilidad de liberar los elementos del flojo normal del documento y posicionarlos a voluntad. Se tiene la posición absoluta, que es la que libera el elemento del flujo normal del texto, la posición relativa, que es la que se define a partir de la posición normal, o inicial, del elemento y no lo libera del flujo de texto, y la posición fija, tiene las mismas condiciones que Absolute.Según Schulz, R. G. (2008).
